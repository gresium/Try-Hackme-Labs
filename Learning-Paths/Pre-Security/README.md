
# Pre Security

**Status:** ✅ Completed

## Overview
The Pre Security path on TryHackMe is the starting point for anyone entering the cybersecurity field. It covers the foundational knowledge needed before diving into offensive or defensive security topics.

## What I Learned
- **Networking Fundamentals** — How the internet works, IP addressing, DNS, HTTP/S, and core protocols
- **How the Web Works** — Understanding web servers, browsers, and how requests/responses function
- **Linux Fundamentals** — Navigating the Linux file system, permissions, processes, and basic commands
- **Windows Fundamentals** — Windows file system, registry, user management, and key security features

## Key Takeaways
This path built the core mental model needed for everything that follows — understanding how systems communicate, how operating systems are structured, and how data flows across networks is essential before touching any offensive or defensive tooling.
